# DSMovie

DSMovie é uma aplicação full stack web construída durante a semana [Spring React](https://devsuperior.com.br).

# Technologies used
## Back end
- Java
- Spring Boot

## Front end
- TypeScript
- ReactJS
- HTML
- CSS

## Implantation in production
- Back end: Heroku
- Front end web: Netlify
- Banco de dados: Postgresql
